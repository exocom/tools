Installing Java JDK on Windows

Test

  • java -version

Download

  • http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html
  • 64 bit if you have 64bit
  • 32bit - Windows x86 123.64 MB jdk-7u51-windows-i586.exe
  • 64bit - Windows x64 125.46 MB jdk-7u51-windows-x64.exe

Install

  • run installer

Test

  • java -version

Installing Java JDK on Mac

Test

  • java -version

Download

  • Mac OS X x64 179.49 MB jdk-7u51-macosx-x64.dmg

Install

  • mount dmg
  • run install

Test

  • java -version

Installing Java JDK on Linux (Debian)

Test

  • java -version

Install

  • sudo add-apt-repository ppa:webupd8team/java
  • sudo apt-get update
  • sudo apt-get install oracle-java7-installer

Test

  • java -version
