Installing Node.js on Windows

Test

  • node --version

Download

  • http://nodejs.org/download/
  • Windows Installer (.msi)
  • 64 bit if you have 64bit

Install

  • run installer

Test

  • node --version

Installing Node.js on Mac

Test

  • node --version

Download

  • http://nodejs.org/download/
  • Mac OS X Installer (.pkg)

Install

  • mount dmg
  • run installer

Test

  • node --version

Installing Node.js on Linux (Debian)

Test

  • node --version

Install

  • sudo apt-get update
  • sudo apt-get install -y python-software-properties python g++ make
  • sudo add-apt-repository ppa:chris-lea/node.js
  • sudo apt-get update
  • sudo apt-get install nodejs

Test

  • node --version
